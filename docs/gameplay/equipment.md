## Thuộc tính phụ theo rank

| Rank | Pool thuộc tính phụ có thể mở |
|---|---|
| Rare (+1) | Crit, Attack Speed, Life Steal, Luck, Bonus Exp, Bonus Farm Yield |
| Epic (+2) | Như Rare + Move Speed, MP Regen, HP Regen |
| Legendary (+3) | Như Epic + AOE Bonus, Skill CD Reduction, Gold Find |
| Mythic (+4) | Như Legendary + Invincibility Frame, Double Strike Chance |

> Thuộc tính phụ được **chọn ngẫu nhiên** khi item drop. Reroll bằng **Cuộn Tái Tạo** (Nhà giả kim).

## Điểm thuộc tính tự do

Mỗi lần lên cấp nhận **10 điểm thuộc tính tự do**.

| Phân bổ | Giá trị |
|---|---|
| 1 điểm → HP | +100 |
| 1 điểm → MP | +100 |
| 1 điểm → ATK | +10 |
| 1 điểm → DEF | +5 |
| 1 điểm → Move Speed | +2 |
| 10 điểm → Crit | +1% |
| 10 điểm → Attack Speed | +1% |
| 10 điểm → Luck | +1 |

> **Gợi ý theo hệ:** Kiếm sĩ → ATK + Crit · Song Kiếm → ATK + Attack Speed · Thương sĩ → DEF + HP · Cung thủ → Luck + Crit · Ninja → Move Speed + Crit

## Cường hóa

Trang bị cường hóa từ **+1** đến **+10** tại Lò rèn.

### Hệ số tăng ATK / DEF

| Mức | Tăng thêm mỗi mức | Tổng tăng so với gốc |
|---|---|---|
| +1 | +3% | +3% |
| +2 | +3% | +6% |
| +3 | +3% | +9% |
| +4 | +5% | +14% |
| +5 | +5% | +19% |
| +6 | +5% | +24% |
| +7 | +8% | +32% |
| +8 | +8% | +40% |
| +9 | +8% | +48% |
| +10 | +15% | **+63%** |

> **Ví dụ:** Kiếm Rare ATK 26 → +10 = **ATK 42**
