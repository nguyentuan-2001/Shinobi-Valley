# Shinobi Valley

> Avatar Teamobi gặp Ninja School trong thế giới Stardew Valley.

---

## Mô tả ngắn

Shinobi Valley là game 2D Pixel Art kết hợp farming sim và action RPG. Người chơi vừa quản lý nông trại ban ngày, vừa chiến đấu và khám phá ban đêm.

## Thông tin kỹ thuật

- **Engine:** Phaser + TypeScript hoặc Godot
- **Art Style:** Pixel Art, Top Down, Sprite 32x32
- **Save:** JSON, Data Driven
- **Thời gian dự kiến:** 6–12 tháng
