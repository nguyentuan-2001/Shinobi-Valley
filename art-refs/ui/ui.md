# UI Icons — AI Image Prompts

**Style chung (dán vào đầu mọi prompt bên dưới) — đúng theo [`art-refs/theme.md`](../theme.md), bắt buộc đọc file đó trước khi gen:**

```
Shinobi Valley art style, cozy fantasy farming game UI, pixel-art inspired, bright natural colors, soft saturation, warm neutral light, slightly stylized, high readability, not realistic, not photorealistic, not overly vibrant, not dark, consistent palette, friendly cute style, clean simple shapes.
Single centered UI element, flat front-facing silhouette, no scene or background elements, no cast shadow unless the prompt says otherwise.
Transparent background, nothing behind the element, no floor, no ground plane, no white/solid backdrop, no stray text (except a symbol the icon itself needs, like X or !).
Designed to remain readable at small HUD sizes (16x16–64x24).
Avoid: dark fantasy, HDR, cinematic lighting, oversaturated colors, neon colors, glossy 3D render, heavy outlines, horror atmosphere, realistic textures, white background.
```

UI cần đặc biệt: **clean, dễ đọc ở kích thước nhỏ, không quá nhiều chi tiết**.
